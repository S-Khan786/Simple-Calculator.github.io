<h1>Calculator App:</h1>

This is a simple calculator app built with HTML, CSS, and JavaScript.

<h2>Features:</h2>

The calculator app supports the following operations:

Addition,
Subtraction,
Multiplication, and
Division

<h2>Usage:</h2>

To use the calculator app, simply enter the numbers you want to calculate using the number buttons on the calculator. Then, click on the operation button (+, -, x, or รท) to perform the calculation. The result will be displayed in the calculator's display area.
